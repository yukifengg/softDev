Shell Scripting

To engage in shell scripting, go into your machine's terminal and follow this tutorial:

  1. Write your own bash by creating [name of youre file].sh file
  2. Find where bash is stored by typing this command: which bash
  3. Once file has been created, nano into it and insert this into the code: #! usr/bin/bash echo12;
  4. Change user with chmod u + x shell.sh
  5. Run ./shell.sh and it will return what you typed in the echo command
  • Everything in bash is string, so if you want to use bash other than a string, you have to expand the variable with the $ symbol
  • All for loops in bash need to start with do and close with done
  • Double brackets give you conditionality
  • Double parantheses allow arithmetic
